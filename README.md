# Cyber-24 UCA Termux Setup
একটি প্রফেশনাল Termux সেটআপ স্ক্রিপ্ট যা আপনার টার্মিনালকে স্টাইলিশ লুক এবং প্রয়োজনীয় সব প্যাকেজ একবারে ইনস্টল করে দেবে।

## বৈশিষ্ট্য (Features)
* স্টাইলিশ **Cyber-24 UCA** রঙিন ব্যানার।
* অটোমেটিক সিস্টেম আপডেট এবং আপগ্রেড।
* প্রয়োজনীয় প্যাকেজ ইনস্টল: Python, Git, PHP, Ruby, এবং আরও অনেক কিছু।
* লাইভ সিস্টেম ইনফো (User, Device, IP) প্রদর্শন।

## কিভাবে ব্যবহার করবেন (Installation)
Termux ওপেন করে নিচের কমান্ডগুলো একে একে দিন:

```bash
pkg install git -y
git clone [https://github.com/b4z1d/Termux---setup-.git](https://github.com/b4z1d/Termux---setup-.git)
cd Termux---setup-
chmod +x setup.sh
./setup.sh
